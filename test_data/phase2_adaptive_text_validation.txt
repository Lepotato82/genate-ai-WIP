=== Known Issues logged for Phase 3 ===
1. Apple-touch-icon black background baked in — visible as black box on light-bg brands
   Root cause: apple-touch-icon files include a background fill that is not transparent
   Fix: Pillow-based background removal / transparency crop before compositing
2. Linear primary_color == background_color — accent_bar invisible
   Root cause: CSS token extraction finds no distinctive brand color variable
   Fix: when primary_color == background_color, fall back to a contrasting hue
      (e.g. use secondary_color, or derive a complementary color from primary)
3. Single-slide output on short-copy runs (Chargebee, Lemon Health)
   Root cause: formatter wrote very short body (no paragraph breaks) → no splits
   Not a bug — copy quality issue, not image_gen issue
